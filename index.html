Explications et contre-mesures

JavaScript sert à rendre les sites web interactifs et non pas à les sécuriser.
Il ne faut donc jamais placer d'informations sensibles dans du code JavaScript côté client.
En effet ce dernier est visible dans le code source de la page.
Pour accéder au code source de la page, appuyez sur CTRL + U.
Il ne faut pas non plus oublier de revérifier côté serveur toutes les données vérifiées en JavaScript.
